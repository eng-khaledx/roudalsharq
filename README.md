# data/cars.json — دليل الإضافة

## Structure كل سيارة

```json
[
  {
    "id": 1,
    "name": "تويوتا برادو",
    "brand": "تويوتا",
    "model": "برادو",
    "year": 2024,
    "price": 185000,
    "interest_rate": 3.99,
    "fuel": "بنزين",
    "transmission": "أوتوماتيك",
    "image": "images/cars/prado.jpg",
    "description": "سيارة دفع رباعي فاخرة"
  }
]
```

## الحقول

| الحقل | النوع | الوصف |
|-------|-------|-------|
| `id` | رقم | معرّف فريد لكل سيارة (لا يتكرر) |
| `name` | نص | الاسم الكامل للسيارة |
| `brand` | نص | الماركة (تويوتا، هيونداي، ...) |
| `model` | نص | الموديل (برادو، كامري، ...) |
| `year` | رقم | سنة الصنع |
| `price` | رقم | السعر بالريال (بدون فواصل) |
| `interest_rate` | رقم | نسبة الفائدة (مثال: 3.99) |
| `fuel` | نص | بنزين / ديزل / هايبرد / كهرباء |
| `transmission` | نص | أوتوماتيك / عادي |
| `image` | نص | مسار الصورة أو رابط خارجي |
| `description` | نص | وصف مختصر للسيارة |

## الصور

ضع صور السيارات في مجلد `images/cars/` وأشر إليها هكذا:
```
"image": "images/cars/اسم_الملف.jpg"
```

أو استخدم رابط خارجي مباشر:
```
"image": "https://example.com/car.jpg"
```
